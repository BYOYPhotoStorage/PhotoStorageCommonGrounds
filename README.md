# Photo Storage - Common Grounds

A space for ideation, planning, and technical design for a direct-to-cloud photo backup app.

## Folder Structure

- **ideas/** — Brainstorming, feature ideas, user flows, rough concepts
- **specs/** — Product specs, engineering specs, requirements documents
- **research/** — Provider comparisons (Backblaze B2, etc.), API investigations, cost analysis
- **architecture/** — System design, data flow diagrams, component breakdowns
- **decisions/** — Architecture Decision Records (ADRs), trade-off analyses, chosen approaches
